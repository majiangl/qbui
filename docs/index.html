BUTTON

可作为按钮使用的元素

按钮类可应用于 a、 button 或 input 元素。

Link 
<a href="javascript:void(0)" class="btn">Link</a>
<button class="btn" type="button">Button</button>
<input class="btn" type="button" value="Input">

禁用状态

a 元素默认不支持 disabled 属性，需要添加 .disabled 类来达到禁用效果。

Link 
<a href="javascript:void(0)" class="btn disabled">Link</a>
<button class="btn" type="button" disabled>Button</button>
<input class="btn" type="button" value="Input" disabled>

INPUT

基本用法

添加 .text-input 类应用样式，使用 disabled 属性禁用输入框。 

<input type="text" class="text-input" value="Normal">
<input type="text" class="text-input" value="Disabled" disabled>

带图标的输入框

可以添加额外的类来增强输入框的显示

<input type="text" class="text-input search">

DATEPICKER

插件官方文档： https://github.com/eternicode/bootstrap-datepicker

基本用法

可以通过 $.fn.datepicker.defaults 设置默认值。 

<input id="date" type="text" class="text-input date">
$('#date').datepicker({
    language: "zh-CN",
    autoclose: true,
    todayHighlight: true,
    daysOfWeekHighlighted: "0,6"
});

取值/设值

<input id="date1" type="text" class="text-input date">
// Get date
$('#date1').datepicker('getDate');
// Get string
$('#date1').val();
// Set date
$('#date1').datepicker('update', '2016-08-08');
// or
$('#date1').datepicker('setDate', '2016-08-08');

事件

所有事件都会在input上触发。下面是 changeDate 事件的示例。 

<input id="date2" type="text" class="text-input date">
$('#date2').on('changeDate', function(evt){
    alert('Event changeDate triggered');
});

SELECT

插件官方文档： https://harvesthq.github.io/chosen/

基本用法

使用原生的 select 初始化插件。 

<select name="speed" id="chosen">
    <option>Slower</option>
    <option>Slow</option>
    <option selected="selected">Medium</option>
    <option>Fast</option>
    <option>Faster</option>
</select>
$("#chosen").chosen({
    width: '200px',
    disable_search: true
});

更新状态

若要对select本身或选项的状态（比如禁用）进行更改，或添加删除选项，需要在原生的select元素上进行修改，通过调用 .trigger("chosen:updated") ，插件自动从原生select更新状态。 

<select name="speed" id="chosen1">
    <option>Slower</option>
    <option>Slow</option>
    <option>Medium</option>
    <option>Fast</option>
    <option>Faster</option>
</select>
<input id="chosen1Btn1" type="button" class="btn" value="Disable">
<input id="chosen1Btn2" type="button" class="btn" value="Enable">
$('#chosen1Btn1').click(function(){
    $('#chosen1').prop('disabled', true).trigger('chosen:updated');
});

$('#chosen1Btn2').click(function(){
    $('#chosen1').prop('disabled', false).trigger('chosen:updated');
});

事件

所有事件在原生select元素上触发。 

<select name="speed" id="chosen2">
    <option>Slower</option>
    <option>Slow</option>
    <option>Medium</option>
    <option>Fast</option>
    <option>Faster</option>
</select>
$('#chosen2').change(function(evt, data){
    alert('Current selected: '+data.selected);
});

AUTOCOMPLETE

插件官方文档： http://jqueryui.com/autocomplete/

基本用法

使用 $.ui.autocomplete.prototype.options 设置默认值。 

<input type="text" id="ac" class="text-input">
var availableTags = [
    "ActionScript",
    "AppleScript",
    "Asp",
    "BASIC",
    "C",
    "C++",
    "Clojure",
    "COBOL",
    "ColdFusion",
    "Erlang",
    "Fortran",
    "Groovy",
    "Haskell",
    "Java",
    "JavaScript",
    "Lisp",
    "Perl",
    "PHP",
    "Python",
    "Ruby",
    "Scala",
    "Scheme"
];
$( "#ac" ).autocomplete({
    source: availableTags,
    autoFocus: true
});

处理数据

Autocomplete在搜索建议选项时，允许用户提供自己的逻辑，简单而又具有灵活性。 

<input type="text" id="ac1" class="text-input">
$('#ac1').autocomplete({
    source: function(request, response){
        var item = 'You searched: '+request.term;
        response([item]);
    }
});

事件

jquery ui有统一的事件绑定机制，以下两种方式均可。 

<input type="text" id="ac2" class="text-input">
$('#ac2').autocomplete({
    source: availableTags,
    select: function(event, ui) {
        alert('select event triggered');
    }
});
// or
// $('#ac2').on('autocompleteselect', function(event, ui){});

TOOLTIP

插件官方文档： http://qtip2.com/

基本用法

默认采用 title 属性的值作为提示内容。使用 $.fn.qtip.defaults 设置默认值。

Hover Me! 
<span id="qtip" title="I'm Tooltip!">Hover Me!</span>
$('#qtip').qtip();

静态提示

编程式设置静态提示。

Click Me! 
<span id="qtip1">Click Me!</span>
$('#qtip1').qtip({
    content: {
        text: 'This text is not from title'
    },
    show: 'click',
    position: {
        my: 'bottom left',
        at: 'top right'
    }
});

ajax提示

使用ajax加载提示内容。分两种情况，一是首次显示时加载内容；而是每次显示都加载内容。

Load content with ajax!
Load content continuously with ajax! 
<span id="qtip2" title="I'm Tooltip!">Load content with ajax!</span>
<br>
<span id="qtip3">Load content continuously with ajax!</span>
$('#qtip2').qtip({
    content: {
        text: function(event, api){
            // send you ajax, call api.set in callback
            setTimeout(function(){
                api.set('content.text', 'Return from ajax');
            }, 500);
            
            return 'Loading...';
        }
    }
});

$('#qtip3').qtip({
    content: {
        text: function(event, api){
            // Return promise object to enable continuously load.
            return $.ajax({
                url: 'data/tooltip-ajax.json',
                cache: false
            })
            .then(function(res){
                // Return fetched text
                return res.text;
            });
        }
    }
});

委托代理

若有几百成千或上万的列表需要用到tooltip插件，直接绑定无疑会产生大量的tooltip实例，严重影响性能。这时可以结合jQuery的委托绑定来实现tooltip的delegate。

Item 1 Item 2 Item 3 
<div id="qtip4">
    <span class="tooltip-delegate">Item 1</span>
    <span class="tooltip-delegate">Item 2</span>
    <span class="tooltip-delegate">Item 3</span>
</div>
$('#qtip4').on('mouseover', '.tooltip-delegate', function(evt){
    $(this).qtip({
        // Make sure the tooltip won't be overridden once created
        overwrite: false,
        content: "I'm delegated.",
        show: {
            // Use the same show event as the one that triggered the event handler
            event: event.type,
            // Show the tooltip as soon as it's bound
            ready: true
        }
    }, event); // Pass through our original event to qTip
});

DATATABLE

使用 $.fn.Datatable.defaults 设置默认值。添加 .hover 类启动hover效果；添加 .nowrap 类使得td内容不会换行。

    lengthChange: false,
    info: false,
    scrollX: true,
    ajax: 'data/datatable.json',
    columns: [
        {
            data:'name',
            title:'Name',
            className: 'dt-head-left'
        },
        {
            data:'position',
            title:'Position',
            className: 'dt-head-left'
        },
        {
            data:'office',
            title:'Office',
            className: 'dt-head-left'
        },
        {
            data:'extn',
            title:'Extn.',
            className: 'dt-head-right dt-body-right'
        },
        {
            data:'start_date',
            title:'Start date',
            className: 'dt-head-right dt-body-right'
        },
        {
            data:'salary',

    language: {
        paginate: {
            previous: '',
